# APTEKA HMS

APTEKA HMS (Homeopathy Management System) is a modern clinic management application designed for APTEKA HOMOEOPATHY.

## Features

- Patient Registration & History
- Electronic Medical Records (EMR)
- Homeopathic Prescription Management
- Billing & Receipts
- Daily & Monthly Analytics
- Follow-up Management
- Expense Tracking
- Dashboard & Reports
- Secure User Authentication
- Cloud Sync
- Responsive Design (Desktop & Mobile)

## Technology Stack

- React + TypeScript
- FastAPI (Python)
- PostgreSQL
- Tailwind CSS
- SQLAlchemy
- JWT Authentication
- Docker
